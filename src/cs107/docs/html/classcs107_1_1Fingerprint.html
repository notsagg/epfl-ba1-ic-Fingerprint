◆ getNeighbours()

static boolean [] cs107.Fingerprint.getNeighbours ( boolean  image[][],
int  row,
int  col 
)
inlinestatic

Returns an array containing the value of the 8 neighbours of the pixel at coordinates (row, col).

The pixels are returned such that their indices corresponds to the following diagram:
----------—
| 7 | 0 | 1 |
----------—
| 6 | _ | 2 |
----------—
| 5 | 4 | 3 |
----------—

If a neighbours is out of bounds of the image, it is considered white.

If the row or the col is out of bounds of the image, the returned value should be null.

Parameters
imagearray containing each pixel's boolean value.
rowthe row of the pixel of interest, must be between 0(included) and image.length(excluded).
colthe column of the pixel of interest, must be between 0(included) and image[row].length(excluded).
Returns
An array containing each neighbours' value.
